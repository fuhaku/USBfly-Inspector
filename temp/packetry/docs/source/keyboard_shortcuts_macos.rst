==========================
Keyboard Shortcuts - macOS
==========================

General
-------

+-----------+-------------------------------------------+
| Keypress  | Command                                   |
+===========+===========================================+
| ``⌘ + O`` | Open capture file                         |
+-----------+-------------------------------------------+
| ``⌘ + E`` | Interrupt loading of a large capture file |
+-----------+-------------------------------------------+
| ``⌘ + S`` | Save current capture to file              |
+-----------+-------------------------------------------+

Live Capture
------------

+-----------+--------------------------+
| Keypress  | Command                  |
+===========+==========================+
| ``⌘ + R`` | Scan for capture devices |
+-----------+--------------------------+
| ``F5``    | Scan for capture devices |
+-----------+--------------------------+
| ``⌘ + B`` | Begin live capture       |
+-----------+--------------------------+
| ``⌘ + E`` | End live capture         |
+-----------+--------------------------+
